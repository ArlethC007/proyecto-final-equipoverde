NailStock

Sistema de Gestión para Ferreterías

Descripción breve

NailStock es un sistema de escritorio diseñado para optimizar la gestión operativa y financiera de ferreterías.
Centraliza en una sola plataforma el control de inventario, ventas, proveedores y reportes financieros.
Su interfaz intuitiva permite mejorar la eficiencia del personal, reducir pérdidas por descontrol de stock y facilitar la toma de decisiones comerciales.

Características principales

Control completo de inventario con alertas de stock bajo.

Punto de venta (POS) integrado con cálculo automático de totales.

Gestión de proveedores y clientes.

Reportes financieros y de ventas.

Interfaz gráfica amigable basada en entorno Windows.

Base de datos local.

Tecnologías utilizadas

Python — Lenguaje principal del sistema.

PyQt6 — Framework para la interfaz gráfica de usuario.

SQLite3 — Base de datos local para almacenamiento de información.
